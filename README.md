# uom-android-development
# Development of Applications for Android Mobile Devices

This repository contains laboratory exercises, assignments, and source code developed during my studies at the **University of Macedonia (PAMAK)**, Department of Applied Informatics.

The course covers the core fundamentals of building native Android applications, ranging from basic UI design to advanced mobile concepts like local databases, asynchronous operations, and hardware utilization.

## 👨‍🏫 Course Instructors
* **Associate Professor Apostolos Ampatzoglou**
* **Dr. Alexandros Karakasidis**

## 🛠️ Technologies & Tools Used
* **Languages:** Java / Kotlin
* **UI Design:** XML (Extensible Markup Language)
* **Development Environment:** Android Studio
* **Build System:** Gradle
* **Minimum SDK Target:** Android API 24+ (or specified per lab)

## 📂 Repository Structure & Curriculum
The project is organized based on the weekly laboratory lectures and topics:

* 📑 **Lecture 01:** Java Crash Course & Fundamentals for Android Development
* 📑 **Lecture 02 & 03:** Introduction to GUI Creation & Basic View Components
* 📑 **Lecture 04 & 05:** Object-Oriented Mobile Development, Activities, and Lifecycle
* 📑 **Lecture 06 & 07:** Local Data Storage & Persistence (SharedPreferences, SQLite / Room)
* 📑 **Lecture 08:** Remote Database Connectivity & Network Operations
* 📑 **Lecture 09:** Multimedia Utilization (Audio, Video, and Camera APIs)
* 📑 **Lecture 10:** Remote Operations via Open Data Web APIs (REST, JSON parsing)
* 📑 **Lecture 12:** Advanced Layouts & Mock Examination Practice

## 🚀 Key Learning Outcomes
By completing these laboratory modules, I have gained hands-on experience in:
- Managing the Android Activity lifecycle and communication via **Intents**.
- Designing responsive user interfaces using **ConstraintLayout**, **LinearLayout**, and **RecyclerView**.
- Handling background threads and asynchronous network requests safely.
- Persisting user data locally and fetching real-world data using external **REST APIs**.
